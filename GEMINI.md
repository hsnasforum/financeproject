# Project Context & Rules

## UI/UX Design System (Highest Priority)
- **UI Pattern Priority**: Always follow the patterns in `@./docs/ui/REFERENCE_UI.md` (Segmented Tabs + Provider Chips + Filter Grid + List) as the highest priority.
- **Tone**: Professional, stable, and clean (Banking style).
- **Background**: Very light gray (`bg-slate-50`).
- **Surface**: White cards (`bg-white`).
- **Primary Accent**: **emerald/green** (e.g., `text-emerald-600`, `border-emerald-500`).
- **Radius**: Large (`rounded-2xl` to `rounded-3xl`).
- **Mobile-First**: Use Card/List Row layouts instead of tables on mobile.
