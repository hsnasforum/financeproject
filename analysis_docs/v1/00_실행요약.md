# 00. 실행 요약

## 저장소 기준 핵심 진단

1. 이 저장소는 단일 가계부 앱이 아니라
   `Planning v2`, `Planning v3`, `Recommend`, `Products Catalog`, `DART`, `Ops/Debug`가 함께 있는 금융 워크스페이스입니다.

2. 가장 큰 문제는 기능 부족이 아니라
   **사용자용 / 실험용 / 운영용 / 개발용 경계가 한 제품 안에서 동시에 보이는 점**입니다.

3. 가장 큰 기회는
   **Planning 결과를 Recommend와 Products로 자연스럽게 연결하는 사용자 흐름**을 완성하는 것입니다.

4. 가장 먼저 필요한 정리는
   - Public/Ops/Labs/Dev 노출 정책
   - Canonical Financial Profile DTO
   - API 응답 envelope
   - QA 회귀 범위 문서화
   입니다.
